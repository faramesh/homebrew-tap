# Faramesh Homebrew Tap

Official Homebrew tap for [Faramesh](https://faramesh.dev) — unified governance plane for AI agents.

## Install

```bash
brew install faramesh/tap/faramesh
```

Or tap first, then install:

```bash
brew tap faramesh/tap
brew install faramesh
```

## Quick start

```bash
faramesh demo           # See governance in action (30 seconds)
faramesh init           # Auto-detect env, generate policy skeleton
faramesh serve --policy policy.yaml   # Start the governance daemon
```

## Upgrade

```bash
brew upgrade faramesh/tap/faramesh
```
